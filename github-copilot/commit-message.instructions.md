---
description: "Use when writing commit messages: English, imperative mood, GitHub emoji code prefix (e.g. :sparkles:, :bug:)"
---

All commit messages must be written in English.

Rules:

- Start with GitHub emoji code (e.g. `:sparkles:`, `:bug:`)
- Do NOT use actual emoji characters
- Use imperative English

Examples:
- `:sparkles: add onboarding screen`
- `:bug: fix crash on launch`
